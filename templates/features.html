Features

What Cadence actually ships.

Every feature designed around the music release window. Not a generic Buffer clone.

✓Multi-account TikTok OAuth

Connect every TikTok in your rollout via TikTok's official OAuth flow with PKCE. Tokens encrypted at rest with AES-256.

✓Native Content Posting API

Direct chunked upload via TikTok's Content Posting API. No scraping. No browser automation. No ban risk.

✓Unified release calendar

Every artist, label, and show account on one calendar. Avoid same-hour collisions during release week.

✓Per-brand caption templates

Show feeds need different copy than artist feeds. Save voice-locked templates per account so the right tone goes out every time.

✓Auto-publish at scheduled time

Background worker fires posts within seconds of the scheduled time. Refresh-token rotation handled automatically. No phone reminders.

✓Cancel before publish

Mistakes happen. Cancel any queued post until it enters the publishing window.

✓Privacy-first storage

OAuth tokens encrypted with Fernet / AES-256. Per-user isolation. Delete-account flow removes everything within 7 days.

✓Status tracking

Every post tracked through queued → publishing → posted or failed. See exactly when something went wrong and why.

✓Built by working production crew

Cadence is built by Bigger Dreams, a working LA production company. Wish You Were Here, our flagship show, was the test customer.
